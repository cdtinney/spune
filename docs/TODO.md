# Spune Roadmap

## Near-term

- [ ] **Visual regression tests** — Playwright screenshot comparisons for key visual states. See [plan](plans/visual-regression-tests.md).

## Medium-term

- [ ] **Richer album discovery** — Deeper artist graph traversal, genre-based suggestions, Spotify recommendations API. See [plan](plans/richer-album-discovery.md).
- [ ] **Ambient mode enhancements** — Configurable animation speed, drift patterns, sleep/dim mode. See [plan](plans/ambient-mode.md).
- [ ] **Song history/queue awareness** — Mosaic evolves based on listening session, not just current track. See [plan](plans/song-history.md).

## Longer-term

- [ ] **Spotify Web Playback SDK** — Use the SDK to receive real-time playback events when Spune is the active device, eliminating polling. Fall back to current polling for external devices. See [plan](plans/spotify-playback-sdk.md).
- [ ] **Multi-user party mode** — Multiple users contributing to the mosaic simultaneously. Spotify Jams has no public API (as of 2026), so this would need a custom room/session system. See [plan](plans/multi-user-party.md).
